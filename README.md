# Vanillin

Vanillin is a JavaScript DOM user interface library.

Try it out at the [playground](http://metaes.org/playground.html).

## Installing

For the latest stable version:

```bash
npm install vanillin
```

For dev builds:

```bash
npm install vanillin@dev
```
